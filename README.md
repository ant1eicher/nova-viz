# nova-viz

One-page D3 data visualizations, generated by Nova and published via GitHub Pages.

- **Live:** https://ant1eicher.github.io/nova-viz/
- Each chart lives under `charts/YYYY-MM-DD-<slug>/index.html`
- D3 v7.9.0 is vendored in `vendor/` — charts are fully self-contained, no CDN.

⚠️ Note: this repo (and the Pages site) is **public**. No sensitive data.
